# Terminal Linux

Este repositorio contiene información y recursos relacionados con la terminal de Linux, enfocándose en las distribuciones Arch Linux y Debian.

## Estructura del Proyecto

```
.
├── Arch/         # Recursos específicos de Arch Linux
└── Debian/       # Recursos específicos de Debian
```

## Contenido

### Arch Linux
Documentación y recursos relacionados con la terminal en Arch Linux, incluyendo:
- Comandos básicos
- Configuración del shell
- Personalización de la terminal
- Scripts útiles

### Debian
Recursos y guías para la terminal en Debian, incluyendo:
- Comandos fundamentales
- Gestión de paquetes
- Configuración del sistema
- Herramientas de terminal

## Uso

Cada directorio contiene documentación específica para cada distribución. Navega a la carpeta correspondiente según tu distribución de interés.
